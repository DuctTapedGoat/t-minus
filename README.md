# T-Minus Optimized Screensaver

A sleek, bouncing countdown timer screensaver for Windows.

## 📥 Downloads

Looking for the screensaver? Just download `T-Minus.scr` above!

## ⚙️ Configuration

The screensaver includes a built-in configuration GUI. To access it:

* **Run the .scr file normally** (Double-click or Install)
* **Press `S`** while the screensaver is running to open settings.
* **Settings change automatically** once you click "SAVE & START".

## 🛠️ Build Your Own (Optional)

If you've modified the source code (`t-minus.py`), you can build a new optimized `.scr` file by running:

1. Open PowerShell in this directory.
2. Run `.\build.bat`.

The optimized build system uses a virtual environment to keep the file size around **16-17MB** instead of the usual 30MB+.
